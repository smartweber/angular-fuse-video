# TrainingTube 2.0 Frontend #

### What is this repository for? ###

* The TrainingTube 2.0 Frontend - based on Angular/Fuse 
* Version 0.1

### How do I get set up? ###

* Clone the repo
* Install ng-cli
* Run `npm install` in repo root
* Dependencies
  * Node
  * Angular CLI
* Run `ng-serve` in repo root
* Open a browser on http://localhost:4200
